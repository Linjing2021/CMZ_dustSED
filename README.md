# CMZ_dustSED
This repository contains the continuum maps and SED-fitting products used in our study of the Central Molecular Zone (https://arxiv.org/abs/2511.20300). It includes all datasets used to produce Figures 1–3 of the paper.

The files correspond to the figures in the manuscript as follows. 

For Figure 1, the dataset includes Cont_0450_micron_CMZ_SCUBA2_stacked_combined.fits (JCMT/SCUBA-2 450 micron continuum) and Cont_0850_micron_CMZ_SCUBA2_stacked_combined.fits (JCMT/SCUBA-2 850 micron continuum). 

Figure 2 uses Cont_0350_micron_CMZ_SHARC2_stacked_combined.fits, the CSO/SHARC-2 350 micron continuum map. 

Figure 3 contains the SED-derived products: CMZ_NH2_8.5_2.5.fits (H2 column-density map, 8.5" resolution), CMZ_dust_Tem_8.5_2.5.fits (dust temperature map, 8.5" resolution), and CMZ_dust_beta_14.0_2.5.fits (dust emissivity index beta map, 14" resolution).
